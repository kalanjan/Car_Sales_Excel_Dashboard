# Car_Sales_Excel_Dashboard
Presented by: Yamsani Kalanjan

Objective
To analyze large-scale car sales data and uncover insights on:
Brand performance
Pricing and mileage trends
Fuel type distribution
The project focuses on transforming raw data into an interactive Excel dashboard that effectively communicates business insights.

| Attribute   | Description                            |
| ----------- | -------------------------------------- |
| **Records** | 50,000 cars                            |
| **Fields**  | Brand, Price, Mileage, Fuel Type       |
| **Source**  | Rhives Technologies Internship Dataset |

| **Category  **           | Observation                                               |
| -------------------- | --------------------------------------------------------- |
| **Top Brands**       | VW and BMW dominate the dataset                           |
| **Fuel Trends**      | Petrol cars are most common; hybrid cars least adopted    |
| **Mileage**          | VW offers highest mileage; Porsche focuses on performance |
| **Price vs Mileage** | Inverse relation ‚Äî higher price, lower mileage            |
